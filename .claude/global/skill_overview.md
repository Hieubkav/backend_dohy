# Tổng quan AI Skills

## API Skills
- **api-cache-invalidation**: Hệ thống tự động invalid cache với Laravel Observers và Next.js on-demand revalidation. Dùng khi cần sync data real-time giữa backend và frontend.
- **api-design-principles**: Các nguyên tắc thiết kế REST và GraphQL API xây dựng interface có khả năng mở rộng, dễ bảo trì.
- **api-documentation-writer**: Tạo tài liệu API toàn diện, OpenAPI specs, và tài liệu cho nhà phát triển.

## Database Skills
- **database-backup**: Luôn backup database trước khi migration, sử dụng Spatie backup, giữ tối đa 10 backup gần nhất cho an toàn.
- **create-skill**: Tạo skill mới cho AI Agent theo đúng cấu trúc chuẩn, kiểm tra trùng lặp và đăng ký vào hệ thống.

## Development Skills
- **docs-seeker**: Tìm kiếm tài liệu kỹ thuật sử dụng llms.txt standards, phân tích GitHub repository, và khám探索 song song nhiều nguồn.
- **seo-expert**: Chuyên gia SEO toàn diện về tối ưu kỹ thuật, Core Web Vitals, E-E-A-T, topic clusters, và nội dung semantic.
- **ui-styling**: Xây dựng UI đẹp với shadcn/ui components, Tailwind CSS, và thiết kế visual canvas cho ứng dụng React.
- **ux-designer**: Hướng dẫn thiết kế UI/UX chuyên nghiệp, tập trung vào giao diện độc đáo, accessibility, và trải nghiệm người dùng.
- **web-performance-audit**: Đánh giá hiệu suất web toàn diện, đo lường tốc độ trang, xác định bottleneck và đề xuất tối ưu hóa.
- **zustand-state-management**: Quản lý state React với Zustand, TypeScript, persistence và devtools middleware.

## Specialized Skills
- **api-documentation-writer**: Tạo tài liệu API chi tiết với endpoint descriptions, examples, authentication guides và error codes.
